This is a font?

Why, yes. Yes it is. These silhouettes can be installed on your computer or used as a web-font. Each silhouette is a real, live person.

More will be added soon. I plan on having one for every letter of the alphabet, as well as numbers and special characters.

How do I use this?

As a font?

Download the font here and install it on your computer. Attached is a readme file that should explain everything. Should...

On the web?

You will be able to install this as a webfont very soon. You can attach the SVG (scalable vector graphic) files within your document and call out their class names. Easy peasy.
